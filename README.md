COMP314 – Advanced Application Design
Assignment 3 – Django Polls Application

This project is based on the official Django tutorial (Parts 1–4).

Features:
- Poll question list
- Voting system
- Results page
- Admin interface
- Custom styling

Setup:
1. python -m venv venv
2. venv\Scripts\activate
3. pip install -r requirements.txt
4. python manage.py migrate
5. python manage.py createsuperuser
6. python manage.py runserver
